# Chapter 8 — Learning, attention, and hierarchical models in continuous state-spaces

Chapter 8 extends generalized filtering by treating model parameters and precision
parameters as hidden variables. The companion implements a compact linear version
of this idea: hidden-state perception, first-order parameter learning, second-order
log-precision learning, and two-layer hierarchical message passing are all driven by
one variational free-energy objective.

## Recipe

| Step | Chapter 8 parameterization |
|---|---|
| 1. Generative process | Observations are generated by `y = x* - theta* + noise`; hierarchy examples use a lower sensory layer and an upper context layer. |
| 2. Generative model | `LearningAttentionModel` predicts observations from `mu_x - mu_theta` and regularizes `mu_x` toward a state attractor. |
| 3. Inference objective | `learning_attention_free_energy` sums precision-weighted errors for sensory data, hidden state, first-order parameter `theta`, and log precision `zeta`. |
| 4. Update rule | `simulate_learning_attention` uses fast state descent and damped slow flows for `mu_theta` and `mu_zeta`. |
| 5. Diagnostics | Figures show state tracking, parameter convergence, learned positive variance, prediction errors, and hierarchical message terms. |
